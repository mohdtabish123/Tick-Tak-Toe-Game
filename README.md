## 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- Two-player gameplay (X vs O)
- Winner detection
- Draw detection
- Restart/New Game button
- Responsive and clean UI
- Real-time game updates

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
